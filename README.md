### ChainForge

A visual prompt-chaining studio inspired by LangChain. Build, visualize, and execute sequential AI workflows using Google Gemma-27B.

Setup

```
git clone https://github.com/ayushs1209/chainforge.git

```

```
cd chainforge

```

```
npm i
npm run dev
```

Navigate to `http://localhost:5173` to access the ChainForge studio.

### Features

- Visual prompt chaining interface
- Integration with Google Gemma-27B
- Save and load prompt chains
- Export chains as reusable templates
